# 🖼️ 2x2 Grid Frame Design Specifications

กรอบรูปแบบ 2x2 Grid เหมาะสำหรับรูปกลุ่มที่จัดวางเป็นตาราง 4 ช่อง

## 📐 มิติและพิกเซล (Dimensions)

*   **ขนาด Canvas ทั้งหมด**: `1965 × 1695 px`
*   **ขนาดช่องรูปภาพ (Photo Hole)**: `900 × 675 px` (อัตราส่วน 4:3)
*   **ระยะขอบด้านข้างและด้านบน (Margin)**: `60 px`
*   **ระยะห่างระหว่างช่องรูปภาพ (Spacing)**: `45 px`
*   **พื้นที่วางลายเซ็นและ QR Code ด้านล่าง**: สูง `240 px` (พิกัด Y = `1455` ถึง `1695 px`)

---

## 📍 พิกัดการเจาะรูรูปภาพ (X, Y Coordinates)

นักออกแบบจะต้องเว้นพื้นที่เหล่านี้ให้โปร่งใส (Transparent) ในไฟล์ PNG 32-bit:

*   **แถวบน (Row 1)**:
    *   **ช่องซ้าย**: X = `60 px` \| Y = `60 px` *(กว้าง 900 x สูง 675 px)*
    *   **ช่องขวา**: X = `1005 px` \| Y = `60 px` *(กว้าง 900 x สูง 675 px)*
*   **แถวล่าง (Row 2)**:
    *   **ช่องซ้าย**: X = `60 px` \| Y = `780 px` *(กว้าง 900 x สูง 675 px)*
    *   **ช่องขวา**: X = `1005 px` \| Y = `780 px` *(กว้าง 900 x สูง 675 px)*
