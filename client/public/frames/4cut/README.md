# 🎞️ 4-Cut Frame Design Specifications

กรอบรูปแบบ 4-Cut เป็นรูปแผ่นยาว 4 ช่องแนวตั้ง ยอดนิยมสไตล์ตู้ถ่ายรูปเกาหลีคลาสสิก

## 📐 มิติและพิกเซล (Dimensions)

*   **ขนาด Canvas ทั้งหมด**: `1200 × 3675 px`
*   **ขนาดช่องรูปภาพ (Photo Hole)**: `1080 × 810 px` (อัตราส่วน 4:3)
*   **ระยะขอบด้านข้างและด้านบน (Margin)**: `60 px`
*   **ระยะห่างระหว่างช่องรูปภาพ (Spacing)**: `45 px`
*   **พื้นที่วางลายเซ็นและ QR Code ด้านล่าง**: สูง `240 px` (พิกัด Y = `3435` ถึง `3675 px`)

---

## 📍 พิกัดการเจาะรูรูปภาพ (X, Y Coordinates)

นักออกแบบจะต้องเว้นพื้นที่เหล่านี้ให้โปร่งใส (Transparent) ในไฟล์ PNG 32-bit:

1.  **ช่องที่ 1**: X = `60 px` \| Y = `60 px` *(กว้าง 1080 x สูง 810 px)*
2.  **ช่องที่ 2**: X = `60 px` \| Y = `915 px` *(กว้าง 1080 x สูง 810 px)*
3.  **ช่องที่ 3**: X = `60 px` \| Y = `1770 px` *(กว้าง 1080 x สูง 810 px)*
4.  **ช่องที่ 4**: X = `60 px` \| Y = `2625 px` *(กว้าง 1080 x สูง 810 px)*
